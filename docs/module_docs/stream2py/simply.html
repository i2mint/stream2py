Helper Function to construct a SourceReader and StreamBuffer in one

stream2py.simply.mk_stream_buffer(read_stream: Callable[[Optional[Any]], Optional[Any]], open_stream: Callable[], Optional[Any]] = None, close_stream: Callable[[Optional[Any]], None] = None, info: Callable[[Optional[Any]], dict] = None, key: Callable[[Any], stream2py.utility.typing_hints.ComparableType] = None, maxlen: Optional[int] = 100, sleep_time_on_read_none_seconds: float = None, auto_drop: bool = True, enumerate_data: bool = True)[source]¶

Helper Function to construct a SourceReader and StreamBuffer in one

from time import sleep
from stream2py import mk_stream_buffer

# download txt from: https://www.gutenberg.org/cache/epub/1524/pg1524.txt
FILE = 'hamlet.txt'
with mk_stream_buffer(
    read_stream=lambda open_inst: next(open_inst, None),
    open_stream=lambda: open(FILE, 'rt'),
    close_stream=lambda open_inst: open_inst.close(),
    key=lambda read_data: read_data[0],
    maxlen=None,
) as stream_buffer:
    buffer_reader = stream_buffer.mk_reader()
    sleep(1)
    # do stuff with buffer_reader
    print(buffer_reader.range(start=40, stop=50))
Parameters

  • read_stream – takes open instance and returns the next read item

  • open_stream – open stream and return an open instance such as a file descriptor

  • close_stream – take open instance and close the instance

  • info – dict help in describing the instance

  • key – sort key for read items

  • maxlen – max read items to keep in buffer window

  • sleep_time_on_read_none_seconds – time to wait if data is not ready to be read

  • auto_drop – False is useful when data can be read independent of time and will quickly overflow the buffer maxlen before data is consumed. StreamBuffer.drop() must be called to continue reading after buffer has filled up. Default True.

  • enumerate_data – wrap read data in an enumerated tuple. Default True.
